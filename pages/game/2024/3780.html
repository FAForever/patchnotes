Patch 3780

Date: Jan 26th 2023

A new year that we can only start with new mechanics that are buggy!

With gratitude to all those who took the time to report issues.

-- Jip

Bug Fixes

Fix the Easy AI not working.

We often use the Easy AI as a testing platform.

Fix a bug where Auto Overcharge appears enabled but is actually disabled.

The bug occurs when the ACU attempts to use Auto Overcharge while enhancing. The Auto Overcharge behavior will be turned off but the UI was not updated accordingly. The UI now reflects this behavior.

Fix a bug when the Billy is redirected by a Loyalist.

The bug would generate an endless loop of explosions.

Features

Give AI a notion of rating

The rating shows in the lobby and in-game. The rating adjusts depending on whether it is a cheating AI, the cheating modifiers and the size of the map. Note that games with AIs are unrated.

Other

  • Improve the English description of various Cybran units
