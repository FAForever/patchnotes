Game Patch 3747

Features

Improve reclaim overview

Adjusts the colors, scaling and depth to better represent what is valuable.

Add toggle to enable / disable always showing splash damage

It is off by default, you can find the toggle in the game options -> interface -> cursor features

Paint feature for observers

Observers can now paint on the map and other observers can view each others paintings. Useful for casting to communicate to your viewers and fellow casters.

Works by pressing the ALT key, can be adjusted in the game options.

This feature also works in replays.

Improve rendering quality of water

Slightly adjusts how water looks like in-game to make it look better overall

Add in missing tooltips for game options

All the game options should now have a consistent tooltip.

Change default aix multiplier values

Especially the AIs that can be found in the vault become complete beasts when they have twice the resources and build time. The default multiplier is now set to 1.5

Note that you can change the AI multipliers in the lobby options

Prefetch session and unit assets

Decreases the loading time when launching through the lobby. Assets of units are loaded as you play to reduce stuttering of the game.

This is an experimental change.

Tweak network parameters

Reduce the amount of lag that players experience by reducing the delay when messages are send.

This is an experimental change.

Bug Fixes

Fix the resource sharing panel

The 'resources' button is now available again in the diplomacy panel

Fix TMDs trying to shoot down missiles that are too nearby

They now have a minimal firing range, it prevents them from trying to hit missiles that they can't find a firing solution regardless

Fix and update recall feature

In particular, fixes a few glitches that people could apply to influence the results.

Slightly reduce mods manager size

Fixes the overlap of the last mod in the list with the search bar

  • Fix weapon priorities not working for units that can snipe
  • Fix the UEF factory unit build animations
  • Fix the cheat window build preview hanging around
  • Fix SubtractCurrentEngineer and formatting in base manager
  • Revert accidental footprint changes of Titan, Loya and Ilsa
  • Scenario Framework GetRandomEntry fix

Other

It would otherwise spam the logs, causing a delay
