Game Patch 3724

Gameplay

  • An alternative approach to loading in custom strategic icons
  • Fix UEF Triad and UEF Destroyer projectile on impact animation

Bug Fixes

  • Fix scathis packing animation time
  • Fix Cybran drone visibility for other players than the owner
  • Fix UI textures being overridden by mods that are not enabled
  • Fix Cybran drone being interactable and other small issues (with thanks to Archsimkat)
  • Fix units being gifted to the same player causing a soft-crash for the shared army mod (co-op campaign)
  • Revert changes to sending the results of games
  • Fix overcharge mouse indicator to use the right damage calculations

Stability

  • Prevent fetching blueprints for potential entities with no blueprints
  • Fix significant hard-crash potential that patch 3721 introduced (with thanks to all the debugger reports)
  • Fix potential soft-crash when gifting units upon death (with thanks to FemtoZetta)
  • Add SCD support for large icon sets (with thanks to Deribus)
  • Revert changes to some projectiles that caused them to crash for mods (with thanks to DDDX)

Other changes

  • Add support for custom game options being set by the server (for 3v3 / 4v4 TMM)
